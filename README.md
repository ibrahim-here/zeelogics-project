# Zeelogics Website

A dark-themed, single-page marketing website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Fonts**: Figtree (headings) & Inter (body)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── layout/       # Header, Footer
│   ├── sections/     # Hero, Projects, About, etc.
│   ├── ui/           # Button, Card, Icon
│   └── shared/       # SectionContainer, AnimatedElement
├── constants/        # Content and configuration
├── types/            # TypeScript interfaces
└── styles/           # Global CSS
```

## Design System

### Colors

- Charcoal: #1A1A1A
- Lighter: #2F2F2F
- Neon Accent: #00FF00 (customizable)

### Typography
- Headings: Figtree
- Body: Inter
